Data Analysis with Python

  • Read dataframe from csv or excel files with pandas.read_csv or pandas.read_excel
  • Write dataframe to csv or excel files with pandas.DataFrame.to_csv or pandas.DataFrame.to_excel
  • Access dataframe rows and columns by pandas.DataFrame.loc or pandas.DataFrame.iloc
  • Access a single value of dataframe by pandas.DataFrame.at or pandas.DataFrame.iat
  • Find, remove or replace null values in a dataframe
  • Concatenate dataframes along index or column by pandas.concat
  • Merge two dataframes by pandas.DataFrame.merge
  • Add a new column to dataframe by pandas.DataFrame.apply
  • Add new column to dataframe groups by pandas.core.groupby.GroupBy.apply
  • Filter rows of dataframe groups by pandas.core.groupby.GroupBy.apply
  • Filter dataframe groups by pandas.core.groupby.DataFrameGroupBy.filter
  • Aggreage over columns of dataframe groups by pandas.core.groupby.DataFrameGroupBy.agg
  • Filter dataframe groups by largest size applying pandas.Series.nlargest
  • Sort dataframe column by the order of a list using pandas.Categorical
  • Fit a curve to data by scipy.optimize.curve_fit
