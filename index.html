Data Analysis with Python

  • Read/Write dataframe from/to csv and excel files
  • Access dataframe rows and columns by pandas loc, iloc, at and iat
  • Merge two dataframes by pandas merge
  • Add a new column to dataframe by pandas apply
  • Add new column to dataframe groups or filter rows of groups by pandas groupby and apply
  • Filter dataframe groups by pandas groupby and filter
  • Get max and min of dataframe groups columns by pandas groupby and agg
  • Filter dataframe groups by largest size applying pandas groupby and nlargest
  • Sort dataframe column by the order of a list using pandas Categorical
  • Fit a curve to data by scipy curve_fit
